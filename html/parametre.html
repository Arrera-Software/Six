Menu principale

Page d'accueil des paramètres de Six qui vous permet de sélection le menu ou se trouve les paramètre que vous voulez modifier :

  • Assistant : Menu qui vous permet de modifier le nom de votre assistant et sa prononciation voir
  • Utilisateur : Menu qui vous permet de modifier le nom que voulez que votre assistant vous appelle voir
  • Internet : Menu qui vous permet de choisir votre moteur de recherche et de donnez vos lien d'agenda et de stockage en ligne voir
  • Application web : Menu qui vous permet d'enregistrer vos application ou site web préférer voir
  • Traducteur : Menu qui vous permet sélection les langues avec qui le logiciel de traduction voir
  • Météo : Menu qui vous permet de modifier les lieus ou vous désirer avoir la météo a l'aide de votre assistant voir
  • Thème GUI : Menu qui vous permet de modifier le thème de l'interface de l'assistant voir
  • Application : Menu qui vous permet d'enregistrer et de sélection le raccourcie de vos applications que vous voulez que votre assistant puisse ouvrir voir
